# Design Reference

이 디렉토리에는 프로젝트의 디자인 참고 자료가 포함되어 있습니다.

## 파일 목록

- **screen.png** - 계산기 UI 디자인 스크린샷
- **code.html** - 완전한 HTML/CSS 구현 참고 코드

## 디자인 특징

- **다크 모드 우선**: 기본 다크 테마
- **모던한 UI**: Space Grotesk 폰트, Material Icons
- **반응형 레이아웃**: 모바일 우선 디자인
- **부드러운 애니메이션**: Hover, Active 상태 전환

## 색상 팔레트

- Primary: `#135bec`
- Background (Dark): `#101622`
- Surface (Dark): `#1e293b`
- Surface Darker: `#111827`

## 사용 방법

`code.html` 파일을 브라우저에서 열어 완전한 디자인을 확인할 수 있습니다.
